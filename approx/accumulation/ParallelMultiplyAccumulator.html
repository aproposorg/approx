    nIn

    the number of parallel input operands

    inAW

    the width of the first input operands

    inBW

    the width of the second input operands

    accW

    the width of the accumulator

    signed

    whether the input operands are signed (defaults to false)

    pipes

    the number of pipeline stages (defaults to 0)

    comp

    whether to use the compressor tree generator (defaults to false)

    targetDevice

    a string indicating the target device (defaults to "", meaning ASIC)

    mtrc

    which metric to use for selecting counters (defaults to efficiency)

    approx

    the targeted approximation styles (defaults to no approximation) Pipelining relies on retiming!
